# Enterprise ETL Modernization

Enterprise migration project demonstrating the modernization of an on-premises data platform (Ab Initio, AutoSys, Teradata) to an AWS-native lakehouse using:

- AWS Glue
- Amazon S3
- Apache Iceberg
- Amazon Athena
- Apache Airflow
- Terraform
- Docker
- PySpark

> This project is being developed incrementally to mirror a real enterprise migration.
